# STM32 综合例程

一些综合运用STM32外设的例程，供学习参考。

项目描述和用到的外设见例程README。

使用[https://github.com/XJU-Hurricane-Team/STM32_Template](https://github.com/XJU-Hurricane-Team/STM32_Template)模板构建。

| 名称                 | 描述              | 是否用操作系统 |
| -------------------- | ----------------- | -------------- |
| uart-record-to-flash | Flash存储串口数据 | N              |
| record-imu-to-flash  | Flash存储IMU数据  | N              |
